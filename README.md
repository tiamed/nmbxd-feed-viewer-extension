# 安装方式
- [下载](https://github.com/tiamed/nmbxd-feed-viewer-extension/releases)压缩包
- 打开 chrome://extensions/
- 打开开发者模式
- 加载已解压的扩展程序

# 使用方式
- 固定本插件
- 右键插件图标
- 点击选项
- 设置订阅ID
- 点击插件图标即可查看订阅
